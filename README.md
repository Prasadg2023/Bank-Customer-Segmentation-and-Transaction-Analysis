# 💳 Bank Customer Segmentation and Transaction Analysis

## 📌 Project Overview

This project focuses on analyzing bank customer data to understand customer behavior, segment users, and derive actionable business insights. Using Power BI, interactive dashboards were created to visualize transaction patterns, customer demographics, and financial trends.

---

## 🎯 Objectives

* Segment customers based on behavior and transactions
* Identify high-value and low-value customers
* Analyze spending patterns and transaction frequency
* Improve decision-making for marketing and customer retention

---

## 📂 Dataset Description

The dataset includes the following key attributes:

* Customer ID
* Age
* Gender
* Account Balance
* Transaction Amount
* Transaction Type
* Transaction Date
* Location

---

## 📊 Key Insights

* High-value customers contribute a major portion of total revenue
* Younger customers show higher transaction frequency
* Certain regions have higher transaction volumes
* Credit transactions dominate over debit in some segments
* Customer segmentation helps in targeted marketing strategies

---

## 📈 Dashboard Features

* Customer Segmentation (High, Medium, Low Value)
* Transaction Trends Over Time
* Region-wise Analysis
* Gender and Age Group Insights
* Transaction Type Breakdown

---

## 🛠️ Tools & Technologies

* Power BI (Data Visualization & Dashboarding)
* Excel / CSV (Data Source)
* Data Cleaning & Transformation (Power Query)

---

## 🚀 How to Use

1. Download the `.pbix` file from the repository
2. Open it using Power BI Desktop
3. Interact with filters, slicers, and visuals to explore insights

---

## 📌 Business Impact

* Helps banks identify profitable customers
* Improves personalized marketing campaigns
* Enhances customer retention strategies
* Supports data-driven decision making
